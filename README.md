## Projeto principal

### [Easy Leilão](https://easyleilao.xyz) — SaaS · 2025–atual

Plataforma que organiza e simplifica a busca de imóveis em leilão da Caixa Econômica Federal. Agrega dados públicos, aplica filtros inteligentes e usa algoritmo próprio para análise de cada imóvel — para que qualquer pessoa consiga tomar uma decisão informada.

```
Stack   Vite · React · TypeScript · Node.js · Express · PostgreSQL · Prisma · Docker · Vercel
```

> Serviço independente de consulta de dados públicos. Não afiliado à Caixa Econômica Federal.
